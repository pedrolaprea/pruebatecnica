Este repositorio contiene la documentación y evidencias del ciclo de pruebas realizado para la Prueba Técnica de QA. El sistema evaluado es una API REST pura desarrollada en Laravel 10.x que gestiona la autenticación de usuarios, perfiles y transferencias de saldo.

Descripción del Sistema

Arquitectura: API REST pura (Solo Backend).
Autenticación: Token-based mediante Laravel Sanctum.
Base de Datos: Soporta múltiples motores.

Resumen de Casos de Prueba (Pass/Fail)
Se ejecutaron pruebas funcionales centradas en la lógica de negocio y seguridad.
